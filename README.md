# ni-win-le-ma

「你赢了吗」在线测试 - 通过问答计算你的赢指数 🏆 | 一个基于选择题的互动网页项目，用于评估用户当前的"赢状态"

## 项目介绍

本项目隶属于日耳蛮赢学（日耳曼赢学）开源计划，致力于探索西方社会的"胜利闭环"现象，深入剖析种姓制度等赢学相关议题。
网站通过一系列精心设计的问题，帮助用户计算个人赢指数，并根据计算结果展示对应的种姓阶级分类。
